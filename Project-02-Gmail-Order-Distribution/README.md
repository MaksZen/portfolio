# Project 02 — Gmail Order Distribution

## 📌 Overview

Автоматическая система распределения заказов, поступающих по электронной почте.

Workflow получает новое письмо из Gmail, извлекает данные заказа, сохраняет его в Google Sheets, анализирует очередь менеджеров, автоматически выбирает наиболее свободного менеджера, отправляет ему заказ в Telegram и обновляет его очередь.

---

## 🎯 Goal

Полностью исключить ручное распределение заказов между менеджерами и автоматизировать процесс обработки входящих заявок.

---

## ⚙️ Features

- Gmail Trigger
- Извлечение данных из письма
- Парсинг информации о заказе
- Добавление нового заказа в Google Sheets
- Получение списка менеджеров
- Анализ текущей нагрузки менеджеров
- Автоматический выбор менеджера по минимальной очереди
- Отправка заказа в Telegram
- Обновление очереди менеджера
- Запись времени последнего заказа

---

## 🔄 Workflow

1. Gmail получает новое письмо.
2. n8n автоматически запускает Workflow.
3. Из письма извлекаются данные заказа.
4. Заказ сохраняется в таблицу Google Sheets.
5. Загружается список менеджеров.
6. JavaScript определяет менеджера с минимальной очередью.
7. Заказ отправляется выбранному менеджеру через Telegram.
8. Очередь менеджера увеличивается.
9. В таблице обновляется дата последнего заказа.
