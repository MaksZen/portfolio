# Project 01 — Order Processing System

## 📌 Overview

Автоматизированная система обработки входящих заказов.

Workflow принимает данные заказа через Webhook, выбирает свободного менеджера по минимальной очереди, обновляет информацию в Google Sheets, отправляет заказ выбранному менеджеру в Telegram, сохраняет заказ в журнал и возвращает ответ клиентской системе.

---

## 🎯 Goal

Автоматизировать полный цикл обработки нового заказа без участия человека.

---

## ⚙️ Features

- Получение заказа через Webhook
- Подготовка входящих данных
- Получение списка менеджеров из Google Sheets
- Автоматический выбор менеджера с минимальной очередью
- Обновление очереди выбранного менеджера
- Отправка заказа в Telegram
- Сохранение заказа в журнал Google Sheets
- Ответ внешней системе через Webhook Response

---

## 🔄 Workflow

1. Внешняя система отправляет заказ в Webhook.
2. Данные проходят предварительную обработку.
3. Загружается список менеджеров.
4. JavaScript определяет менеджера с минимальной очередью.
5. Обновляется очередь выбранного менеджера.
6. Заказ отправляется менеджеру в Telegram.
7. Заказ сохраняется в журнал Google Sheets.
8. Клиент получает успешный ответ от Webhook.
